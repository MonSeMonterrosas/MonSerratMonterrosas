Ilustración Análoga

° ° ° ° ° °

Es el tipo “clásico” de ilustración, en el que se dibuja y pinta a mano. Las técnicas que se pueden encontrar para este tipo de ilustración son: dibujo, tinta, rotuladores, lápices de colores, pastel, acuarela, entre otros, cuando se utilizan varias técnicas juntas se le llama “técnica mixta”.
